Redux

 Redux is a predictable state container for writing applications that act consistently.
  • Installation and Configuration
    • Install redux: npm install redux react-redux
    • Install reduxjs/toolkit, a package that simplifies using redux: npm install @reduxjs/toolkit
  • Create a simple store
    • Create a store:
      // store.js
import { createStore } from 'redux';

const initialState = {
    count: 0,
};

const reducer = (state = initialState, action) => {
    switch (action.type) {
    case 'INCREMENT':

        return { count: state.count + 1 };
    case 'DECREMENT':
        return { count: state.count - 1 };
    default:
        return state;
    }
};

const store = createStore(reducer);                                            

export default store;
    • Use the store in your app:
      button onClick={() => store.dispatch({ type: 'INCREMENT' })}>+
      button onClick={() => store.dispatch({ type: 'DECREMENT' })}>-
