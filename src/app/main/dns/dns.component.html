  • Definitions
    Domain Name
    the name of a website, e.g. nehsa.net
    Domain Name System (DNS)
    the system that translates domain names to IP addresses
    Top Level Domain (TLD)
    the last part of a domain name, e.g. .com
    Second Level Domain (SLD)
    the part of a domain name before the TLD, e.g. nehsa.net
    Subdomain
    a domain that is part of a larger domain, e.g. blog.nehsa.net
    A Record
    an address record that maps a domain name to an IP address. e.g. "nehsa.net" -> 1.2.3.4
    CNAME Record
    an alias record that maps a domain name to another domain name. e.g. I was folk that go to "www.nehsa.net" -> "nehsa.net"
    DNS providers
    It's common nowadays to use a third party to manage your DNS records. There are several advantages for this:
    • They have a global network of DNS servers, which can help with performance.
    • They can provide additional services, such as DDoS protection.
    • They can provide a web interface for managing your DNS records.
    • They (often) will auto-renew your SSL certificates.


    Example: AzureDNS, Cloudflare, AWS Route 53
