Microsoft Monetize Now is a program that helps you monetize your app.
  • Setup
    • Go to the Microsoft Monetize Now website
    • Sign up for an account
    • Downloads the ads.txt file and place it in the root of your website
    • Configure your website to display it: inside angular.json, add the following to the assets array: "src/ads.txt"
    • Visit the Microsoft Monetize Now website and click on the "Verify" button
