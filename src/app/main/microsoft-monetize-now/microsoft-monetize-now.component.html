Definitions:
Microsoft Monetize Now
Microsoft Monetize Now is a program that helps you monetize your app.
Ads.txt
Ads.txt is not referring to a sales ad but is an acronym for: Authorized Digital Sellers. It is a file that publishers can create to list who is authorized to sell their inventory. It is intended to prevent unauthorized inventory sales.
Ad Unit
Ad Unit is a container for ads. You can have multiple ad units on a page. Each ad unit has a unique ID.
Header Tag
The header tag is a script that goes in the header of your website. It is used to load the Microsoft Monetize Now SDK.
Body Tag
The body tag is a script that goes in the body of your website. It is used to render the ad.
Native Ad
A native ad is an ad that is designed to look like the content it is placed in. It is less intrusive than a display ad.
Display Ad
A display ad is a traditional ad that is displayed on a website. It is more intrusive than a native ad. The only reason to choose a display ad over a native ad is if you wish to annoy your users.

  • Tips
    • How do we create ads of the appropriate size? The Microsoft Monetize Now SDK is supposed to automatically adjust the ad size to fit the container in which it is placed. That's why no options are presented when setting up a new ad unit for size. Still, this didn't work for me, and the ad bled onto the element below it, causing ugliness. I set the container's width to 100% to ensure the ad fits the container and has a fixed height and set the overflow to hidden. This seems to do the trick:

      e.g. 
      <div style="width: 100%; height: 100px; overflow: hidden;" id="<THE ID OF THE ELEMENT YOU WISH THE AD TO SHOW>"></div>
