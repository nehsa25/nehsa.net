VI IMproved
I have always liked VI Improvedvim, maybe because of it's complete lack of effort on a good UX. It's nostalgic to me that VIM hasn't changed since I first used it years ago. Unfortunately, I am not good with it so this is my reference.

  • Commands To enter a command in VIM, press ESC and then type the command. Here are some commands:

    :wq
    Save and quit
    :q!
    Quit without saving
    :q
    Quit
    :w
    Save
    :w filename
    Save as filename
    d, ENTER (or dd)
    Delete a line
    u
    Undo
    y
    Copy (yanky)
    p
    Paste
    Show Color Schemes
    Run :colorscheme and then press tab
    o
    Insert a new line below the cursor
    O
    Insert a new line above the cursor
    0
    Move to the beginning of the line
    $
    Move to the end of the line
    ^
    Move to the first non-whitespace character of the line
    gg
    Move to the beginning of the file
    G
    Move to the end of the file
    /{pattern}
    Search for a pattern. e.g. "/hello" would find the first instance of "hello" from the cursor moving down.
    n
    Move to the next instance of the pattern
    N
    Move to the previous instance of the pattern
    :set number
    Show line numbers
  • .vimrc The .vimrc file is a configuration file for VIM. It is located in the home directory. By default, it does not exist but creating one offers some value:
    • Set the number of spaces to use for tabs
    • Set the color scheme
    • Set the default indentation
    • Set the default syntax highlighting
    • Set to show line numbers
    • Set the default search behavior
    • Lots of other things I'm sure..


    Here's mine. I know, a thing of beauty:


    " Set the number of spaces to use for tabs
set tabstop=4

" Set the number of spaces to use for indentation
set shiftwidth=4

" Set the default indentation to spaces
set expandtab " (use "set noexpandtab" to use tabs)

" Set the color scheme (to get more schemes, run :colorscheme and then press tab)
colorscheme delek " (or whatever scheme you like)

" Set the default syntax highlighting
syntax on

" Show line numbers
set number

" Set syntax based on filetype
au BufReadPost *.conf set syntax=dosini
au BufReadPost *.py set syntax=python
au BufReadPost *.sh set syntax=sh
au BufReadPost *.c set syntax=c
au BufReadPost *.cpp set syntax=cpp
au BufReadPost *.h set syntax=c
au BufReadPost *.hpp set syntax=cpp
au BufReadPost *.java set syntax=java
au BufReadPost *.html set syntax=html
au BufReadPost *.css set syntax=css
au BufReadPost *.js set syntax=javascript
au BufReadPost *.php set syntax=php
au BufReadPost *.sql set syntax=sql
au BufReadPost *.xml set syntax=xml
au BufReadPost *.yml set syntax=yaml
au BufReadPost *.yaml set syntax=yaml
au BufReadPost *.md set syntax=markdown
au BufReadPost *.rst set syntax=rst
au BufReadPost *.txt set syntax=txt
au BufReadPost *.log set syntax=log
au BufReadPost *.json set syntax=json


    Pro-tip:
    • The .vimrc file is read when VIM starts up. If you make changes to the file, you can either restart VIM or run the command :source ~/.vimrc to reload the file.
    • This will not work when running sudo vim, to fix this, run sudo vim -u ~/.vimrc instead or if it won't affect others, place the settings in the /etc/vim/vimrc.local file to make it system-wide.
