  • CSS Animations Developer's log 77886.2, I've been working on a CSS animation for the past few days and awhile back. This is what I've discovered:
    • The easiest way to work with anitmations in the shorthand "animation" CSS property.
      .bob-text {
padding: 1em;
width: 400px;
text-align: center;
color: beige;
z-index: 200;
animation: fadeOut 10s ease-out forwards;
}
      animation: fadeOut 10s ease-out forwards; (animation: name duration timing-function delay iteration-count direction fill-mode play-state;)
      fadeOut
      The name of the CSS animation function. In this case, it keeps something visible fully (no gradual disappearance) until the last 10% of the animation duration, then fades out.
      @keyframes fadeOut {
0% {
    opacity: 1;
}

90% {
    opacity: 1;
}

100% {
    opacity: 0;
}
}
      10s
      The duration of the animation. In this case, it takes 10 seconds to complete.
      ease-out
      The timing function. In this case, it starts fast and slows down at the end. Other options include:
      ease-in
      Starts slow and speeds up at the end
      ease-in-out
      Starts slow, speeds up, then slows down at the end
      linear
      Constant speed
      step-start
      Starts at the beginning of each step then jumps to the end
      step-end
      Starts at the end of each step then jumps to the beginning
      steps(4, end)
      4 steps, each starting at the end of the previous step
      cubic-bezier(0.1, 0.7, 1.0, 0.1)
      Custom cubic-bezier function

      cubic-bezier(x1, y1, x2, y2):
      Defines a curve based on four control points.
      (x1, y1) is the control point for the starting end of the curve.
      (x2, y2) is the control point for the ending end of the curve.
      Both (x1, y1) and (x2, y2) must be between 0 and 1.
      example:
      green/red
      The slow start and end simulate the compression and decompression of the ball, while the quick middle represents the bounce itself.
      animation: bounce 5s infinite cubic-bezier(0.25, 0.8, 0.25, 0.2);
      orange/blue
      Slow start but rapid acceleration
      animation: bounce 5s infinite cubic-bezier(0.4, 0, 0.2, 1);
      yellow/purple
      Object is decelerating for the stop.
      animation: bounce 5s infinite cubic-bezier(0.4, 0.1, 0.8, 1);
      fuchsia/teal
      Elastic/springy type effect
      animation: bounce 5s infinite cubic-bezier(0.1, 1, 0.9, 0);
      forwards
      The number of times the animation will repeat. In this case, it will repeat infinitely. Other options include:
      1
      Runs once
      2
      Runs twice
      forwards
      Runs once and then stops at the end state
      backwards
      Runs once and then stops at the beginning state
      infinitely
      Runs forever
    • To make an animation permanent, meaning the state will not revert back once the animation is complete, use "forwards": animation: fadeOut 10s ease-out forwards;

  • Favicon
    • Get your image:
      (Google Gemini was used to create this Australian Cattle Dog)
    • Take the image to a transparency removal site to create a transparent .PNG file
    • Take that images to an Online FavIcon generator.
      • Resize in in 3 sizes:
        • 16x16
        • 32x32
        • 64x64
    • Add the link to your css. In most cases, theses file need to be in the root of the website but Angular needs them in the asset folder: 
      <link rel="icon" type="image/x-icon" sizes="32x32" href="assets/images/favicon-32x32.png">
<link rel="icon" type="image/x-icon" sizes="16x16" href="assets/images/favicon-16x16.png">
<link rel="icon" type="image/x-icon" sizes="64x64" href="assets/images/favicon.ico">
