    • Working with colors

      • Create a new colors.scss file for a central spot for all our coloring needs. This should be at the same level as styles.scss
        This file should contain: 
        •     // main theme palettes
    $colorful-palette: map.get($color-config, 'colorful');
    $vampire-palette: map.get($color-config, 'vampire');
    $angel-palette: map.get($color-config, 'angel');
    $current-theme-palette: $colorful-palette;
    
    // main colors
    $theme-border-color: mat.get-color-from-palette($current-theme-palette, 500);
    $theme-backround-color: mat.get-color-from-palette($current-theme-palette, 50);
    
    // these variables can be accessed in the global scope
    :root {
        --theme-border-color: #{$theme-border-color};
        --theme-backround-color: #{$theme-background-color-info};
    }

  • Favicon
    • Get your image:
      (Google Gemini was used to create this Australian Cattle Dog)
    • Take the image to a transparency removal site to create a transparent .PNG file
    • Take that images to an Online FavIcon generator.
      • Resize in in 3 sizes:
        • 16x16
        • 32x32
        • 64x64
    • Add the link to your css. In most cases, theses file need to be in the root of the website but Angular needs them in the asset folder: 
          <link rel="icon" type="image/x-icon" sizes="32x32" href="assets/images/favicon-32x32.png">
    <link rel="icon" type="image/x-icon" sizes="16x16" href="assets/images/favicon-16x16.png">
    <link rel="icon" type="image/x-icon" sizes="64x64" href="assets/images/favicon.ico">

  • Selecting Elements
    • If you wish to affect all elements that have a certain class (e.g. all span elements that have the class "code"): 
              span.code {
            background-color: rebeccapurple !important;
            color: silver;
            font-family: Consolas;
            border-radius: 5px;
            padding: 3px 1em;
            font-size: 1rem;
            min-width: 400px;
            max-width: 90%;
            line-height: 5lvh !important;
        }
    • Multiple classes can be specified via comma: 
                  h1, h2, h3, h4, h5, h6 {
                color: plum;
            }
    • Hierarchy can be indicated using (>): 
                  
      CSS:
            div {
                background-color: teal;
            }
            .blah>div {
                background-color: salmon;
            }
                                            
      
                                            
      HTML:
            <div class="blah">
                <div>I'm Salmon!</div>
            </div>
            <div>I'm Teal!</div>
      Example:
      I'm Salmon!
      I'm Teal!
