  • Troubleshooting
    • VSCode is removing import statements upon save or appears like the file is in readonly mode
      The setting "editor.formatOnSave" may be causing this issue; if enabled, it will format the file upon saving, and some languages automatically manage imports, such as GoLang, which employs the VSCode extension "goimports," which automatically manages imports based on usage.
