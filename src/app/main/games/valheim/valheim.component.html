Valheim
Play on nehsa.net server: vh.nehsa.net:2456, password: playnehsa

  • Configure Server
    • Get the Docker image: docker pull lloesche/valheim-server
    • Run it:
      docker run -d \
    --name valheim-server \
    --cap-add=sys_nice \
    --stop-timeout 120 \
    -p 2456-2457:2456-2457/udp \
    -v $HOME/valheim-server/config:/config \
    -v $HOME/valheim-server/data:/opt/valheim \
    -e SERVER_NAME="<SERVER NAME>" \
    -e WORLD_NAME="<WORLD NAME>" \
    -e SERVER_PASS="<SECRET>" \
    lloesche/valheim-server
    • Define A record in DNS for <vh.nehsa.net>
    • Open port in local firewall
    • Port forwarding: <IP>:2456-2458 (Both TCP/UPD)
    • Connect to the server at <vh.nehsa.net>:2456
    • BE A VIKING!
  • Setup Admins
    • Watch from the server as the player you wish to make admin joins your server
    • Get the player's Steam64ID. This will just be a number: XXXXXXXXXXXXXXXX
    • Add it it the adminlist.txt file in the config directory
    • I did not need to restart the server but it certainly wouldn't hurt.
    • Admin commands are now included in the output when you type /help in the conversation window. Further development commands can be enabled by typing: devcommands in the conversation window. This allows for things like teleporting to players: pos <playername>
    • Launch client with -console flag
  • Allow all items through portals In an effort to enforce a sort of progression, by default the game does not allow items such as metals to be transported through portals. This forces the player to explore the world and find the items.

    All well and good the first playthrough..

    If you want to allow all items to be transported through portals, you can do this:
    setworldmodifier portals casual
