Semantic UI

 Semantic UI is a development framework that aims to create responsive layouts using human-friendly HTML.
  • Installation and Configuration I'm not sure all this is needed but it's what I did:
    1. Install semantic-ui-vue, the official Semantic UI integration for Vue.js: npm install semantic-ui-vue
    2. Install semantic-ui-css, the actual CSS for Semantic UI: npm install semantic-ui-css
    Both are needed, as semantic-ui-vue is just the Vue.js integration and does not include the CSS. e.g. it allows for the use of semantic-ui-vue components in your Vue.js app, but does not include the CSS to style them.
