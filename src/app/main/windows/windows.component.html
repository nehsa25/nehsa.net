Windows
  • Configure WSL Linux in Windows!
    • Installation / Configuration
      1. To install Ubuntu: wsl --install
        Other distros can be installed with
        1. wsl --install -d <DISTRIBUTION>
          To list all available distros: wsl --list --online
    • Switching between WSL and back
      • To switch to WSL: wsl

        Multiple WSL environments may be installed. For example, Docker Desktop creates a WSL environment. If you type wsl and go into the wrong environment:
        • wsl --list (-l): See what environments are on the machine
        • wsl -d <DISTRIBUTION>: Specify the environment to use
      • To switch back to Windows: exit
    • Copying Files
      • WSL to Windows: cp /mnt/c/Path/To/File /Path/To/Destination
        e.g. from home to c: would be: cp ~/file /mnt/c/file
      • Windows to WSL: cp /mnt/c/Path/To/File /Path/To/Destination
        e.g. from home to c: would be: cp c:\file ~/
