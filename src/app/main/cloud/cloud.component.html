Using the cloud is more than simply using someone else's computer. Cloud providers offer many services, including extra security and services that would be hard for a small company to provide on its own.
* You'll fully understand the importance of security and a decentralized infrastructure when foreign hackers encrypt every file on every machine in your local office and demand a ransom in cryptocurrency before handing over the decryption tool.
Here's a breakdown of the three major cloud providers and some key differents their offerings:
Provider Services Cost Market Adoption Strong CLI support, tool names
AWS EC2, S3, RDS, Lambda, DynamoDB, etc. Pay as you go, free tier Most widely used aws-cli, boto3, CloudFormation
Azure VMs, Blob Storage, SQL Server, Functions, CosmosDB, etc. Pay as you go, free tier 2nd most widely used az, Azure CLI, ARM templates
Google Cloud Platform Compute Engine, Cloud Storage, Cloud SQL, Cloud Functions, etc. Pay as you go, free tier 3rd most widely used gcloud, Cloud SDK, Deployment Manager


More information about services:
AWS
is the most widely used and has the most services. (e.g. AWS services: 175, GCP: 60, and Azure: 50)
Azure
Azure is the second most widely used and has the best integration with Windows, thanks to features like Active Directory.
Google Cloud Platform
Google Cloud Platform is the third most widely used and has the best integration with Kubernetes (because Google created Kubernetes.)
EC2
is a type of service that allows you to create virtual machines. GPC and Azure offer similar services.
S3
is a type of storage used to store unstructured data like images, videos, and backups. (akin to Azure Blob storage or Google Cloud Storage's buckets)
Blob storage
is a type of storage that is used to store unstructured data like images, videos, and backups.
RDS
is a service that allows you to create a relational database. (akin to Azure SQL Database or Google Cloud SQL)
Lambda
enables you to run code without provisioning or managing servers. (akin to Azure Functions or Google Cloud Functions)
DynamoDB
is a service that will allow you to create a NoSQL database. (akin to Azure CosmosDB or Google Cloud Datastore)
CosmosDB
is a service that will enable you to create a NoSQL database. IN SPACE!
VPC
VPC is a service that allows you to create a virtual private cloud. A VPC is needed when you need more security, isolation, or connection to an on-premise network. (akin to Azure Virtual Network or Google Cloud VPC)
