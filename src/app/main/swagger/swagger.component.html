Swagger Documentation for this website (OpenApi 3.0)

Useful Tips

  • Auto-generation in Kestrel

            var builder = WebApplication.CreateBuilder(args);
        builder.Services.AddEndpointsApiExplorer();
        builder.Services.AddSwaggerGen();    
        var app = builder.Build();
        app.UseSwagger();
        app.UseSwaggerUI();

  • Install the Swashbuckle.AspNetCore.Annotations package
    dotnet add package Swashbuckle.AspNetCore.Annotations

    For annotated as such: 
            app.MapGet("/api/v1/da", [SwaggerOperation(
            Summary = "Returns a single quote",
            Description = "A random quote is returned each time a request is made")]
            [SwaggerResponse(200, "Success")]
            [SwaggerResponse(500, "An error occurred")] () =>
        {
        
            return JsonSerializer.Serialize<string>(quotes[Random.Shared.Next(quotes.Length)]);
        })
        .WithName("DouglasAdamQuotes")
        .WithOpenApi();
