Microsoft Clarity
Microsoft Clarity is an anlytics platform that helps you understand how users interact with your website.
  • Overview Microsoft Clarity is a free analytics platform that helps you understand how users interact with your website. It came out in 2020 and is a competitor to Google Analytics.

    Comparision and how to enable cookieless tracking on both platforms:
    Google Analytic Strengths Google Analytics Cookieless Support w/Examples Clarity Strengths Clarity Cookieless Support w/Examples
    Google Analytics is the most popular analytics platform Cookieless tracking example:
    window.dataLayer = window.dataLayer || [];
function gtag() { dataLayer.push(arguments); }
gtag('js', new Date());
gtag('config', 'GA_MEASUREMENT_ID', { 'anonymize_ip': true });

  • Setup
    • Create an account and obtain your tag
    • Place the tag in the head of your website
    • Modify it as follows to enable cookieless tracking: 
      (function (c, l, a, r, i, t, y) {
