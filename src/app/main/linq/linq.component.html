Language Integrated Querylinq, baby?

 Standardized query syntax in C#.
  • Overview Language Integrated Querylinq is a query technology for .NET. It extends C# with native language syntax for queries across various objects - databases, json, ect.

    It can be used in two forms:

    declarative syntax:
    var query = from item in collection
where item.property == value
select item;
     or, method syntax: var query = collection.Where(item => item.property == value);

    I generally prefer the method syntax because it is more concise and easier to read. Someone once told me to consider the "=>" to be "such that." Doing so makes it much easier to read.

    e.g. I want all items that are from the Canada:

    OK:

    Give me all items "such that" item.Country is equal to "CA":
    var query = collection.Where(item => item.Country == "CA");

    Operation Usage Example Result
    Where Filters a sequence of values based on a predicate. var gastonPossee = persons.Where(person => person.GangId == 2); gastonPossee is a collection of all the people in Gaston's possee.
    Select Projects each element of a sequence into a new form. var names = persons.Select(person => person.Name); names is a collection of all the names of the people.
    OrderBy / OrderByDescending Sorts the elements of a sequence based on a key. var ordered = persons.OrderBy(person => person.Name); ordered is a collection of all the people sorted by name.
    GroupBy Groups the elements of a sequence based on a key. var grouped = persons.GroupBy(person => person.GangId); grouped is a collection of all the people grouped by gang.
    Join Joins two sequences based on a key. var joined = persons.Join(gangs, person => person.GangId, gang => gang.Id, (person, gang) => new { person.Name, gang.Name }); joined is a collection of all the people and their gang names.
    Any Determines if any element of a sequence satisfies a condition. var any = persons.Any(person => person.Name == "Timothée"); any is a boolean indicating if there is a person named Timothée.
    Count Returns the number of elements in a sequence. var count = persons.Count(); count is the number of people.
    Sum Computes the sum of a sequence of numeric values. var sum = persons.Sum(person => person.GangId); sum is the sum of all the gang ids.
    Average Computes the average of a sequence of numeric values. var average = persons.Average(person => person.GangId); average is the average of all the gang ids.
    First / FirstOrDefault Returns the first element of a sequence. var first = persons.First(); first is the first person.
    Single / SingleOrDefault Returns the only element of a sequence, or a default value if the sequence is empty. var single = persons.Single(person => person.Name == "Timothée"); single is the person named Timothée.
    Take Returns a specified number of contiguous elements from the start of a sequence. var take = persons.Take(5); take is the first five people.
    Skip Bypasses a specified number of elements in a sequence and then returns the remaining elements. var skip = persons.Skip(5); skip is all the people after the first five.
    Contains Determines whether a sequence contains a specified element. var contains = persons.Contains(person); contains is a boolean indicating if the person is in the collection.
    Distinct Returns distinct elements from a sequence. var distinct = persons.Distinct(); distinct is a collection of all the unique people.
    Union Combines two sequences and returns distinct elements.

    Intersect Returns the intersection of two sequences. var intersect = persons.Intersect(otherPersons); intersect is a collection of all the people that are in both collections.
    Except Returns the difference between two sequences. var except = persons.Except(otherPersons); except is a collection of all the people that are in the first collection but not the second.
    Reverse Reverses the order of elements in a sequence. var reverse = persons.Reverse(); reverse is a collection of all the people in reverse order.
    Concat Concatenates two sequences. var concat = persons.Concat(otherPersons); concat is a collection of all the people from both collections.
  • .AsQueryAble() A memory friendly version of IEnumerable. The power of .AsQueryAble() is that it allows you to query a large collection without having to load the entire collection into memory. This is useful when you are working with a large dataset and only need a small subset of the data. The query is only executed when an operation like ToList() or ToArray() is called or when the query is otherwise enumerated.

    var query = collection.AsQueryAble().Where(item => item.Property == value);
