NPM

Useful Commands

npm install -g npm@latest - Update NPM (Node Package Manager) to latest

npm update -g npm - Update Node to latest

npm install - Installs packages specified in packages.json

npm audit fix - Fixes easy security problems found in audit (excluding braeaking changes) ng add @angular/material - Installs Material

Packaging a new NPM package

  1. Sign up for an NPM account at npmjs.com
  2. Create folder structure:
    • <packagename>
      • package
        • <The actual contents of the package>
      • test-folder
