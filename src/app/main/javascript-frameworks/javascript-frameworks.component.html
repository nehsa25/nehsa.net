  • Definitions
    ECMAScript 6ES6
    ES6 is the sixth version of the ECMAScript standard, which is the standard on which JavaScript is based. ES6 is a significant update to the language, and the first update to the language since ES5 was standardized in 2009. Implementation of these features in major JavaScript engines is underway now. The latest version is ES11 although ES6 is still widely used.
    Dependency Injection
    Tree shaking is a form of dead code elimination. The term is commonly used in the JavaScript context for the removal of unused code. Tree shaking relies on the static structure of ES6 module syntax (import/export) to determine which modules are used and which are not. This is important for performance, as it reduces the size of the JavaScript bundle that is sent to the client.
  • Web Development Frameworks Designing an application capable of the load you intend to place on it is not just a task but a pivotal decision that can make or break the success of your project and the user experience it delivers.

    When starting a new project, the frameworks that will be used to structure our code must be given great consideration. This is important not only for maintainability but also, in some cases, for performance.
