Bootstrap
  • Overview Popular front-end styling framework. It was created by Twitter and is now maintained by the open-source community.

    Very comparable to Material Design and Tailwind (moreso Tailwind than Material due to the utility class nature (text-center)).

    Example, if you wish to have important text you wish to make prominent, add classes:

    <div class="display-1 text-center">
    Gubernatorial<br>/go͞obərnə'tôrēəl/
</div>
    == Gubernatorial
    /go͞obərnə'tôrēəl/
