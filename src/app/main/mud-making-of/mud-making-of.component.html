August, 2024 - Implementing a web scraping feature


The idea is that within NehsaMUD, if you visit the Wizard, he will have "magic mirrors" which will allow you to pull data from any website and display it in the mud as a "prophecy."
  • checkMicroservice to handle scrapping (Go, gin api) A GoLing API will be created to handle web scraping. This will be a RESTful API that uses the library "goquery" to scrape the data. Goquery is nice because it is much like jQuery and allows you to use CSS selectors to find your desired data. The license is from MIT. This will be a microservice called by NehsaMUD and this page.

    Implementation details can be found here: Go in the "Web scraper" section.
