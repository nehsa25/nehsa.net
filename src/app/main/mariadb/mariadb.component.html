Maria DB
MariaDB is a community-developed, commercially supported fork of the MySQL relational database management system, intended to remain free and open-source.

As the Procedural Language/Structured Query LanguagePL/SQL of MariaDB and MYSQL syntax are the same, this page will focus on the installation and configuration of MariaDB only.

  • Installation
    • Get Docker image: docker pull mariadb:latest
    • Run Docker container: docker run --name <DB-NAME> -p 3306:3306 -v d:\data\mariadb:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=<PASSWORD> -d mariadb:latest
