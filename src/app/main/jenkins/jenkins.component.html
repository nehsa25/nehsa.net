Jenkins

Docker Installation: (https://www.jenkins.io/doc/book/installing/docker/)

  • Create bridge
    docker network create jenkins
  • Create dockerfile 
        FROM jenkins/jenkins:2.440.3-jdk17
    USER root
    RUN apt-get update && apt-get install -y lsb-release
    RUN curl -fsSLo /usr/share/keyrings/docker-archive-keyring.asc \
        https://download.docker.com/linux/debian/gpg
    RUN echo "deb [arch=$(dpkg --print-architecture) \
        signed-by=/usr/share/keyrings/docker-archive-keyring.asc] \
        https://download.docker.com/linux/debian \
        $(lsb_release -cs) stable" > /etc/apt/sources.list.d/docker.list
    RUN apt-get update && apt-get install -y docker-ce-cli
    USER jenkins
    RUN jenkins-plugin-cli --plugins "blueocean docker-workflow"
  • Build Docker image from it
    docker build -t lp-blueocean:2.440.3-1 .
  • Run Image
    docker run --name jenkins-ci-e2e --restart=on-failure -u root --network jenkins --publish 8080:8080 --publish 50000:50000 --volume jenkins-data:/var/jenkins_home lp-blueocean:2.440.3-1
  • Initial Setup from Root Bash
    docker exec -it -u root jenkins-ci-e2e bash
    apt-get install mercurial
    apt-get install vim
    vim /etc/group
    // Add "jenkins" to Jenkins group
    http://localhost:5000/restart
  • Access via localhost:8080
  • Finish configurating in UI

Continuous Integration Example

  • Pull down Jenkins Docker Image/Run it
  • Configure Mercurial (or your SCM)
  • Configure Node
  • Cnfigure new Pipeline Job 
        pipeline {
        agent any
        stages {
            stage ('Deploy Docker image for testing') {
                steps {
                    <>
                }
            }
            
            stage ('Checkout') {
                steps {
                    checkout([$class: 'MercurialSCM', credentialsId: '4cbca40f-4ac7-4dd0-9fc8-2461bbe56644', revision: '2024-Sprint3', source: 'https://<SCM URL>/Code/<PROJECT>'])
                }
            }
    
            stage ('Install Node') {
                steps {
                    nodejs(nodeJSInstallationName: 'Node') {
                        sh 'npm config ls'
                    }
                }
            }
            
            stage('Execute Playwright (e2e-ui) tests') {
                steps {
                        nodejs(nodeJSInstallationName: 'Node') {
                        sh 'npm install playwright'
                        sh 'npm install @playwright/test'
                        sh 'npx playwright install-deps'
                        sh 'npx playwright install'
                        sh 'npx playwright test'
                    }
                }
            }
            
    
            stage('Execute Pytest (e2e-api) tests') {
                steps {
                    sh 'pip install -r requirements.txt'
                    sh 'pytest'
                }
            }
            
            stage('Merge with dev') {
                steps {
                    <>
                }
            }
        }
    }
